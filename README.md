# ShowInfo

**ShowInfo**는 포트폴리오 프로젝트를 보관하고 관리하기 위해 React로 만든 웹사이트입니다.  
작은 프로젝트들도 잊지 않고 한 곳에 문서화하여 저장할 수 있도록 설계되었습니다.

---

## 📋 주요 특징

- **프로젝트 문서화**: 모든 프로젝트를 체계적으로 보관하고 관리.
- **작은 프로젝트의 중요성 강조**: 잊혀질 수 있는 작은 프로젝트들도 기록하고 저장.

---

## 🛠️ 기술 스택

- **프로그래밍 언어**: JavaScript
- **프레임워크**: React
- **스타일링**: CSS
- **배포 플랫폼**: Vercel
